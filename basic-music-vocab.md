## 🎶 音乐基础词汇表 (Basic Music Vocabulary)



### I. 音乐类型 (Music Types/Genres)



| **英文 (English)**   | **中文 (Chinese)** | **解释/描述 (Explanation)**                                  |
| -------------------- | ------------------ | ------------------------------------------------------------ |
| **Classical Music**  | 古典音乐           | 指源自欧洲传统，通常有复杂结构和管弦乐编曲的音乐。           |
| **Pop Music**        | 流行音乐           | 旨在吸引广大受众、商业化的、易于传唱的音乐。                 |
| **Rock Music**       | 摇滚乐             | 强调电吉他、贝斯、鼓和强劲节奏的音乐。                       |
| **Jazz**             | 爵士乐             | 源自非裔美国人社区，以即兴演奏、复调节奏和蓝调音阶为特征。   |
| **Blues**            | 蓝调               | 源自非裔美国人，以三行诗歌、呼应和声和忧郁情感为特征。       |
| **Folk Music**       | 民谣/民间音乐      | 代表特定文化或社区的传统音乐，通常以简单旋律和原声乐器为主。 |
| **Electronic Music** | 电子音乐           | 使用电子乐器、合成器或电脑制作的音乐。                       |
| **Hip-Hop / Rap**    | 嘻哈/说唱          | 包含节奏性强的朗诵（说唱）和打碟采样等元素的音乐文化。       |

### III. 音乐要素 (Musical Elements)



| **英文 (English)** | **中文 (Chinese)** | **解释/描述 (Explanation)**                                  |
| ------------------ | ------------------ | ------------------------------------------------------------ |
| **Rhythm**         | 节奏               | 音乐在时间上的组织，由拍子和时值构成。                       |
| **Melody**         | 旋律               | 一系列音高按照特定节奏组织起来形成的线条，是音乐的主题。     |
| **Harmony**        | 和声               | 多个不同的音高同时发声，它们之间的关系和组合。               |
| **Tempo**          | 速度               | 音乐的快慢，常用**BPM** (Beats Per Minute) 衡量。            |
| **Key**            | 调性               | 音乐围绕一个中心音（主音）及其音阶组织起来的体系。分为**Major** (大调) 和 **Minor** (小调)。 |
| **Timbre**         | 音色               | 乐器或人声的独特声音特质。                                   |
| **Dynamics**       | 强弱/力度          | 音乐声音的大小。常见的术语有 **Forte** (强) 和 **Piano** (弱)。 |
| **Pitch**          | 音高               | 声音的频率，决定了音符的高低。                               |



### IV. 表演与人员 (Performance and Personnel)



| **英文 (English)** | **中文 (Chinese)** | **解释/描述 (Explanation)**                              |
| ------------------ | ------------------ | -------------------------------------------------------- |
| **Score**          | 总谱               | 音乐作品所有声部和乐器记谱的完整版本。                   |
| **Conductor**      | 指挥               | 负责领导和协调管弦乐团或合唱团的人。                     |
| **Orchestra**      | 管弦乐团           | 演奏古典音乐的大型乐团，包含弦乐、木管、铜管和打击乐器。 |
| **Chamber Music**  | 室内乐             | 由少数独奏乐器演奏的古典音乐（如三重奏、四重奏）。       |
| **Improvisation**  | 即兴演奏           | 在表演过程中创作新的旋律或乐段。                         |
| **Solo**           | 独奏/独唱          | 仅由一个表演者或乐器完成的乐段。                         |
| **Chorus / Choir** | 合唱团             | 一群人一起唱歌的组织。                                   |
| **Arrangement**    | 编曲               | 将音乐作品改编为不同乐器、人声或风格进行演奏。           |
